# 外观模式

# 一、介绍

  1. 为子系统中的一组接口提供一个高层接口
  2. 使用者使用这个高层接口

# 二、场景

function bindEvent(elem,type,selector,fn){
    if(fn ===null){
        fn = selector
        selector = null
    }
}

//调用
bindEvent(elem,'click','#div1',fn)
bindEvent(elem,'click',fn)

# 三、总结

  1. 不符合单一职责和封闭开放原则，谨慎使用，不滥用
